# Group-Thomas-Greville

## Creating the environment

You can create the environment by  running:

	conda env create -f environment.yml        

Then activate the environment by running:

	conda activate Group-thomas-greville

Finally, install the ipython kernel by running:

	python -m ipykernel install --user --name Group-thomas-greville --display-name "IPython - Group-thomas-greville"
